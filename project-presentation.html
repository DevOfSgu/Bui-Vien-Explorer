Section 04

Core User Journey (4 Giai Doan)

Giai doan 1: Smart Initialization

Tao SessionId an danh, nap AppSettings tu SQLite, kiem tra local data, trigger background sync, pre-warm location de giam latency.

Giai doan 2: Home Tab - Kham pha Tour

Man hinh Home hien danh sach Routes/Tours. Nguoi dung chon tour va bam Start Tour de vao ban do chi tiet.

Giai doan 3: Tour Map Execution

Render Zones len map, lay vi tri user lien tuc voi Stream + Polling fallback, tinh Haversine va kich hoat Hysteresis + Debounce.

Giai doan 4: Detail, Saved, Settings

Mo POI detail, kiem tra ShopHours, luu favorite local-first vao GuestFavorites, sau do worker dong bo server khi co mang.

Section 05

Thong So Ky Thuat Loi

Advanced Location Engine

Engine 1
Stream + Polling Fallback Uu tien LocationChanged event. Neu qua 10s khong co event thi kich hoat Polling de tranh UI bi dong bang.
Engine 2
Noise Filtering + Debounce Loai bo diem co Accuracy > 50m. Chi xac nhan vao zone khi user dung lien tuc trong ban kinh suot 3 giay.
Engine 3
Hysteresis Anti Ping-Pong Enter radius = 15m, Exit radius = 20m de tranh nhay qua lai giua 2 POI sat nhau.
Location Parameters:
  • Polling fallback trigger: 10 giay
  • Accuracy threshold: 50 met
  • Debounce confirmation: 3 giay
  • Hysteresis: Enter 15m / Exit 20m

Offline-First Sync Architecture

Source of Truth tren Mobile

  • SQLite la nguon du lieu render UI
  • LocalZone, LocalRoute, LocalFavorite
  • Khong render truc tiep tu API

Favorite Sync Worker

  • Save local ngay lap tuc (<100ms)
  • Day vao queue dong bo
  • Server 200 OK - danh dau da sync

Data Refresh Worker

  • Fetch Routes/Zones/ShopHours
  • Replace SQLite theo lo
  • Update UI qua binding/event
