Kiến Trúc Tổng Quan

Kiến trúc 3-tier với ASP.NET Core 10 backend, .NET MAUI mobile app, và shared library. Backend sử dụng SQL Server, mobile sử dụng SQLite offline-first.

Backend Stack

  • ASP.NET Core 10.0 Web Framework
  • Entity Framework Core 10 (ORM)
  • SQL Server (Production)
  • MVC Controllers + Services pattern
  • Cookie-based Authentication (Admin + Vendor dual-scheme)

Mobile Stack

  • .NET MAUI (Cross-platform UI)
  • Targets: Android, iOS, macOS, Windows
  • SQLite (Local offline database)
  • MVVM Pattern with Xaml
  • HttpClient for API communication

Shared Library

  • Shared data models (LocalZone, LocalRoute, etc.)
  • SqliteConnectionFactory
  • Entity validation & annotations

Chapter 04

Mobile App Workflow

Startup Flow — 4 Phases

Pha 1 — App Initialize

MauiProgram.cs khởi tạo DI containers, register services, pages, viewmodels. Setup fonts (Space Grotesk), logging.

Pha 2 — Language Selection

Check AppSetting['SelectedLanguage']. Nếu null → Show LanguageSelectionPage. User chọn ngôn ngữ (vi, en, fr...) → Save to SQLite.

Pha 3 — Data Sync (Parallel)

Đồng thời: getOfflineZones() từ SQLite (0ms), fetch từ API, request GPS permissions, load narrations theo ngôn ngữ chọn.

Pha 4 — Main Screen Ready

Map renders tất cả POI, GPS marker hiển thị vị trí user, geofence monitoring xong → app ready phát audio.

Chapter 05

GPS & Geofencing

Xử lý vị trí GPS, kiểm tra zone entry/exit, trigger narration tự động.

3 Giai Đoạn Xử Lý (Planned)

Giai Đoạn 1
GPS Collection (LocationService) watchPosition() liên tục → throttle 5s → tránh tính toán liên tục, tối ưu pin. Update chấm vị trí trên map.
Giai Đoạn 2
Zone Detection (GeofenceService) Tính Haversine distance từ user → mỗi POI. Nếu ≤ radius (default 30m) → pending entry. Ra ngoài → cooldown 5 phút (tránh spam).
Giai Đoạn 3
Audio Decision (NarrationService) Debounce 3 giây → confirm user really here → chọn POI ưu tiên (priority score) → queue for playback.

Haversine Distance Calculation

private double CalculateHaversine(double lat1, double lon1, double lat2, double lon2) { const double R = 6371000; // Earth radius (meters) var dLat = ToRad(lat2 - lat1); var dLon = ToRad(lon2 - lon1); var a = Math.Sin(dLat / 2) * Math.Sin(dLat / 2) + Math.Cos(ToRad(lat1)) * Math.Cos(ToRad(lat2)) * Math.Sin(dLon / 2) * Math.Sin(dLon / 2); var c = 2 * Math.Asin(Math.Sqrt(a)); return R * c; // Distance in meters }
Key Constants:
  • GPS Throttle: 5 seconds
  • Geofence Debounce: 3 seconds
  • Default Radius: 30 meters
  • Cooldown Period: 5 minutes

Chapter 06

Audio Narration (Planned)

2-Tier Audio Strategy

Tier 1
Pre-generated MP3 (0ms) POI có AudioUrl đã sẵn → fetch trực tiếp, phát ngay. Cached locally.
Tier 2
Local Text-to-Speech (1-3s) Fallback nếu không có audio file. Sử dụng TextToSpeech.SpeakAsync() built-in API (iOS/Android native).

NarrationService (Planned)

PlayNarrationAsync(zone)

  • Get narration theo language
  • Fallback chain: target → en → vi
  • Tier 1: Try AudioUrl
  • Tier 2: Fallback TTS

Stop playback

  • Stop audio file playback
  • Cancel TTS
  • Clear notification
Feature: Phát audio tự động khi user vào zone, tự dừng khi ra khỏi zone hoặc có notification/call đến.

Chapter 07

Offline-First Architecture

Mobile app hoạt động 100% offline với SQLite cache. Sync khi có mạng.

4 Lớp Phòng Ngự

Lớp 1 — SQLite Cache

Mỗi lần open app: check LocalZone table → hiển thị ngay (0ms). Auto-load từ API nếu rỗng.

Lớp 2 — Auto Sync

Khi online: check LastSyncTime. Nếu > 6 hours → auto-sync toàn bộ data. Retry với exponential backoff nếu fail.

Lớp 3 — Language Fallback

Narration fallback chain: target lang → English → Vietnamese. Never show blank screen.

Lớp 4 — Offline Analytics

LocalAnalytics log activity → batch upload khi online. No data loss.

User Experience: User có thể dùng app hoàn toàn offline. Dữ liệu sync tự động khi có kết nối. Analytics được ghi lại cả khi offline.

Admin Dashboard & Vendor Portal

Backend cung cấp 2 admin areas với authentication riêng biệt.

Authentication Scheme

// Program.cs setup builder.Services.AddAuthentication() .AddCookie("AdminAuth", options => { options.LoginPath = "/Admin/Auth/Login"; options.ExpireTimeSpan = TimeSpan.FromHours(8); options.Cookie.Name = "AdminAuthCookie"; }) .AddCookie("VendorAuth", options => { options.LoginPath = "/Vendor/Auth/Login"; options.ExpireTimeSpan = TimeSpan.FromHours(8); options.Cookie.Name = "VendorAuthCookie"; });

Features

👨‍💼 Admin Dashboard

  • Routes Management: CRUD
  • Zones (POI): Create/Edit/Delete, set geofence
  • Narrations: Manage scripts, audio status
  • Analytics: Dashboard, heatmap
  • User Management

🏪 Vendor Portal

  • View own POIs only
  • Edit description (không được xóa/thêm)
  • View own analytics
  • Request audio regeneration
